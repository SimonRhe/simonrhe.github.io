Why I chose static site generator Eleventy for my blog

Why use a static site generator

I had worked hard to master HTML, CSS and vanilla JavaScript, and so when the time came to set up my personal blog, I was looking for a tool that would give me 100% control over the blog‚Äôs code. I wanted a tool that would allow me to keep the code clean, simple and lightning fast, while helping me automate the more mundane tasks.

Specifically, I wanted to automate:

  • creating an HTML file for each blog post
  • referencing the blog posts from the home page
  • categorizing the posts using tags
  • uploading the files to the server.

My blog, for now at least, doesn‚Äôt need to access live databases, provide user sign in or other more advanced features, and so a static site generator‚Äîa piece of backend software that can combine text and Markdown files with HTML templates to produce static HTML pages‚Äîwas what I needed.

Eleventy vs. other static site generators

There are many static site generators out there: Hugo, Next.js, Jekyll, and Gatsby are the most popular. I ultimately settled on Eleventy, an up-and-coming static site generator started in 2017, for three main reasons:

  1. Eleventy gives me 100% control over the resulting HTML code and does not bloat it with libraries and structures that my simple blog site will not use
  2. Eleventy is written in JavaScript‚Äîa language I know well‚Äîand runs on Node.js, which allows me to customize it easily and use existing JavaScript libraries to generate HTML pages
  3. Eleventy is simple, so I don‚Äôt have to learn a new framework just to get my site going.

As I started using Eleventy, however, I realized that there are also a few drawbacks to using it:

  • There are relatively few existing templates that we can use when compared to other static site generators
  • The documentation is patchy at times and for some features, I found myself having to look at other people‚Äôs code on GitHub to figure out how Eleventy works.

Nevertheless, although the learning curve was steeper than I expected, Eleventy does what it promises and it is easily extensible by using plugins made specifically for Eleventy, or calling JavaScript libraries.
